# 프로젝트 블루프린트

## 개요

HTML, CSS, JavaScript를 사용하여 Firebase Authentication 및 Realtime Database와 연동된 로그인/회원가입 시스템을 단일 `index.html` 파일 내에 구축합니다. 초기 랜딩 페이지와 모달 기반의 로그인/회원가입 UI를 제공하며, 사용자 친화적인 에러 메시지와 UI 편의 기능을 제공합니다.

## 기능

*   **초기 화면 (랜딩 페이지)**
*   **로그인 모달**:
    *   '로그인' 버튼 클릭 시 모달 대화상자가 나타납니다.
    *   **모달 우측 상단에 닫기(X) 버튼이 추가됩니다.**
*   **회원가입 모달**:
    *   로그인 모달 내의 '회원가입' 버튼 클릭 시 회원가입 모달이 나타납니다.
*   **Firebase 연동**:
    *   Authentication 및 Realtime Database 기능은 그대로 유지됩니다.
*   **동적 UI**:
    *   로그인/로그아웃 상태에 따라 UI가 동적으로 변경됩니다.
*   **에러 처리**:
    *   회원가입 시 발생하는 에러를 한글 메시지로 변환하여 안내합니다.

## 현재 작업

로그인 모달 창에 닫기(X) 버튼을 추가하고 관련 스타일 및 기능을 구현합니다.

### 할 일

1.  `login-dialog` 내부에 닫기 버튼을 위한 `<span>` 태그를 추가합니다.
2.  닫기 버튼에 대한 CSS 스타일(위치, 모양, hover 효과)을 추가합니다.
3.  `login-dialog`에 `position: relative` 속성을 부여합니다.
4.  닫기 버튼 클릭 시 `login-dialog`가 닫히도록 JavaScript 이벤트 리스너를 추가합니다.
5.  수정된 `index.html` 파일을 저장합니다.
6.  변경 사항을 깃허브에 커밋하고 푸시합니다.
