# 프로젝트 블루프린트

## 개요

HTML, CSS, JavaScript를 사용하여 Firebase Authentication 및 Realtime Database와 연동된 로그인/회원가입 시스템을 단일 `index.html` 파일 내에 구축합니다. 초기 랜딩 페이지와 모달 기반의 로그인/회원가입 UI를 제공합니다.

## 기능

*   **초기 화면 (랜딩 페이지)**:
    *   초기 진입 시 '로그인 테스트' 제목과 '로그인' 버튼만 표시됩니다.
*   **로그인 모달**:
    *   '로그인' 버튼 클릭 시, 아이디/비밀번호 입력 필드와 실제 '로그인'/'회원가입' 버튼이 포함된 모달 대화상자가 나타납니다.
*   **회원가입 모달**:
    *   로그인 모달 내의 '회원가입' 버튼 클릭 시, 기존의 회원가입 모달이 로그인 모달 위에 표시됩니다.
*   **Firebase 연동**:
    *   Authentication (로그인, 회원가입, 로그아웃) 및 Realtime Database (회원가입 시 정보 저장) 기능은 그대로 유지됩니다.
*   **동적 UI**:
    *   로그인 성공 시 랜딩 페이지가 숨겨지고 환영 메시지와 '로그아웃' 버튼이 표시됩니다.
    *   로그아웃 시 다시 초기 랜딩 페이지가 표시됩니다.

## 현재 작업

기존 로그인 폼을 모달 대화상자로 변경하고, 초기 랜딩 페이지 UI를 구성합니다.

### 할 일

1.  기존 로그인 폼을 `<dialog id="login-dialog">` 안으로 이동시킵니다.
2.  새로운 초기 화면(제목과 '로그인 모달 열기' 버튼)을 구성합니다.
3.  '로그인 모달 열기' 버튼에 대한 JavaScript 이벤트 리스너를 추가합니다.
4.  로그인 상태에 따라 초기 화면과 환영 메시지 화면이 전환되도록 `onAuthStateChanged` 로직을 수정합니다.
5.  수정된 코드를 깃허브에 커밋하고 푸시합니다.
